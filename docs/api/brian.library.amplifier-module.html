Function Details

voltage_clamp(V, Vc, I, Rs=0.0 ohm, gain=1.0 S, tau_Id=10.0 us, capa_comp=0.0 F)

  

Continuous voltage-clamp.
V = membrane potential (or electrode) variable
Vc = command potential
I = injected current (= recording)
Rs = series resistance compensation
gain [optional] = gain of the voltage-clamp
tau_Id = delay of current feedback (for stability)
capa_comp = capacitance neutralization (not implemented yet)

Continuous current-clamp.
V = membrane potential (or electrode) variable
I = injected current
Vr = recorded potential
Ic = command current
bridge = bridge resistance compensation
capa_comp = capacitance neutralization
