                           MultiLinearStateUpdater


A StateUpdater with one differential equation for each neuron

Initialise with:

``A``, ``B``
    Arrays, see below.
    
The update step for neuron i is S[:,i] = dot(A[:,:,i], S[:,i]) + B[:, i].
