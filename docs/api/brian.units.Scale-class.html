Class Scale

        Scale

Stores the scale factor for each SI dimension.

Probably would only be very rarely used by a user, but might
conceivably be useful in certain circumstances.

scale_factor(self, dim)
Returns the scaling factor for dimension dim For example, if the scale factor of length is milli, and the dimensions of dim are length^2 then the scale factor will be 0.001^2.
 
unit_representation(self, dim)
Returns a representation of the dimension dim at this scale For example, if the scale factor of length is milli, and the dimensions of dim are length^2 then this will return mm^2.
