Function Details

offline_AEC(v, i, ke)

  

Active Electrode Compensation, done offline.
v = recorded potential
i = injected current
ke = electrode kernel
Returns the compensated potential.

correlations(v, i, ksize)

  

Calculates the correlation vector <v(n)i(n-k)>
and the autocorrelation vector <i(n)i(n-k)>
from the recording v and the current i.
ksize is the size of the full kernel (typically 150-200 points)
Returns (vi,ii) (two vectors).

raw_kernel(vi, ii)

  

Calculates the raw kernel from the autocorrelation of the input current
II and correlation between current and recorded potential VI
i.e. finds the best linear filter which transforms I into V.
N.B.: we do not use the Levinson-Durbin algorithm here, which would be
faster (but needs to be implemented in C)

electrode_kernel(Karg, start_tail, dt=0.1 ms, online=False, first_guess=False)

  

Extracts the electrode kernel Ke from the raw kernel K
by removing the membrane kernel, estimated from the
indexes >= start_tail of the raw kernel.
If online is True, then the first two steps are considered
as the feedback delay of the system.

Returns Ke,Km (electrode filter, membrane filter)

kernel_from_step(V, I, dt=0.1 ms)

  

Calculates the (full) kernel from the response (V) to a step input
(I, constant).
If the timestep dt is specified, then the output is in resistance/time.
