Class SpikeContainer



object --+
         |
        SpikeContainer


An object that stores previous spikes.
S[0] is an array of the last spikes (neuron indexes).
S[1] is an array with the spikes at time t-dt, etc.
S[0:50] contains all spikes in last 50 bins.
