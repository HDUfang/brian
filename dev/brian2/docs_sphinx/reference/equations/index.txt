Equations and code strings
==========================

Most of the time, the user does not explicitly construct CodeString or Equations
objects. Instead, directly pass a string to the constructor of NeuronGroup, 
Synapses, etc. The following classes are only useful if you want to have full
control over the namespace in which the code strings are interpreted.
