Preferences
===========

.. data:: brian_prefs

	The Brian preferences object. Access values via attributes, e.g.
	``brian_prefs.prefname``. The preferences object is of type
	:class:`brian2.preferences.BrianGlobalPreferences` and new preferences must
	be defined via the method
	:meth:`brian2.preferences.BrianGlobalPreferences.define`.
